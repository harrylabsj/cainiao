# Cainiao Network（菜鸟物流）

菜鸟物流一站式助手：查包裹、找驿站、估运费、比服务

## What it does
- 菜鸟跨境/国内包裹的物流轨迹查询与状态解读
- 菜鸟驿站取件码查询、位置指引与超期保管规则提醒
- 菜鸟国际物流不同服务等级（超级经济/标准/优先）的时效与费用估算

## Example scenarios

**1. 淘宝海淘包裹追踪**
> 👤 我在淘宝海淘的包裹，菜鸟单号 CN1234567890 到哪了？
> 🤖 显示跨境包裹全链路追踪：中国仓→国际运输→清关状态→最后一公里，解释菜鸟作为物流聚合平台的承运角色

**2. 菜鸟驿站取件查询**
> 👤 我的快递放到菜鸟驿站了，怎么查取件码？
> 🤖 引导通过菜鸟App、淘宝App或短信通知查取件码，说明漏看通知时如何通过订单详情联系驿站，以及驿站营业时间

**3. 菜鸟裹裹查所有包裹**
> 👤 帮我看看我最近一周的菜鸟裹裹包裹列表，有没有一个从义乌发过来的到了驿站还没取？
> 🤖 列出菜鸟账号下近7天所有在途/待取/已签收包裹，筛选义乌发出包裹。如已到驿站提醒取件码和位置（兔喜超市、妈妈驿站等），以及超期保管规则
